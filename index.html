¿Qué es una Pseudo-clase?

Es una palabra clave añadida a un selector que especifica un estado especial del elemento seleccionado.

selector:pseudo-clase {
  propiedad: valor;
}

/* Ejemplo: Cuando el usuario pasa el mouse */
button:hover {
  background-color: blue;
}

Estados de Interacción: :hover

Se activa cuando el puntero del usuario se coloca sobre un elemento. Fundamental para botones y enlaces.

.tarjeta {
  transition: transform 0.3s;
}

.tarjeta:hover {
  transform: translateY(-10px);
  box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}
3 / 16

Estados de Interacción: :active

Se activa en el momento exacto en que se hace clic (mientras se mantiene presionado el botón del mouse).

button:active {
  /* Efecto de presionar el botón */
  transform: scale(0.98);
  background-color: darkblue;
}

Estados de Interacción: :focus

CRUCIAL para la accesibilidad. Se activa cuando seleccionas un input o navegas con el teclado (Tab).

input:focus {
  outline: 2px solid #4fc3f7;
  background-color: #f0f8ff;
}

/* Nunca elimines el outline sin dar una alternativa */
5 / 16

Enlaces: La regla LVHA

Para estilizar enlaces correctamente, el orden en CSS importa: Link, Visited, Hover, Active.

a:link    { color: blue; }   /* No visitado */
a:visited { color: purple; } /* Ya visitado */
a:hover   { color: red; }    /* Mouse encima */
a:active  { color: orange; } /* Click */

Estructurales: :first y :last

Seleccionan elementos según su posición en el padre.

/* El primer párrafo de un artículo */
p:first-child {
  font-weight: bold;
}

/* El último elemento de una lista */
li:last-child {
  border-bottom: none;
}
7 / 16

Estructurales: :nth-child()

La navaja suiza. Selecciona elementos basados en una fórmula matemática.

li:nth-child(2)   { color: red; } /* Solo el 2do */
li:nth-child(odd) { color: blue; } /* Impares (1,3,5) */
li:nth-child(even){ color: green; } /* Pares (2,4,6) */

/* Cada 3 elementos */
li:nth-child(3n)  { margin-bottom: 20px; }

Formularios: :checked

Se aplica a radio buttons o checkboxes cuando están marcados. Permite crear interruptores personalizados.

input[type="checkbox"]:checked + label {
  text-decoration: line-through;
  color: gray;
}

/* Ideal para listas de tareas (To-Do Lists) */
9 / 16

Formularios: :disabled

Estiliza elementos que el usuario no puede usar (inputs desactivados).

button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  background-color: #ccc;
}
10 / 16

Lógica: :not()

La pseudo-clase de negación. Selecciona todo lo que NO cumpla la condición.

/* Todos los botones MENOS el de clase .especial */
button:not(.especial) {
  background-color: gray;
}

/* Párrafos que no son el primero */
p:not(:first-child) {
  margin-top: 1em;
}
11 / 16

El Selector Padre: :has()

¡Revolucionario! Selecciona un elemento padre SI contiene cierto hijo. (Soporte moderno).

/* Selecciona la tarjeta SI tiene una imagen dentro */
.tarjeta:has(img) {
  padding: 0;
}

/* Estila el label si el input de al lado es inválido */
label:has(+ input:invalid) {
  color: red;
}
12 / 16

Agrupación: :is() y :where()

Permiten escribir selectores más limpios y evitar repeticiones.

/* Antes */
header h1, header h2, header p { ... }

/* Ahora con :is() */
header :is(h1, h2, p) {
  color: navy;
}

/* :where() es igual pero con Especificidad 0 */
13 / 16

Validación: :valid e :invalid

Feedback visual instantáneo en formularios.

input[type="email"]:invalid {
  border-color: red;
}

input[type="email"]:valid {
  border-color: green;
}
14 / 16

Pseudo-clases vs Pseudo-elementos

No confundir. Las pseudo-clases usan un : (estado). Los pseudo-elementos usan :: (partes del DOM).

/* Pseudo-clase (Estado) */
div:hover { ... }

/* Pseudo-elemento (Parte virtual) */
div::before { ... }
p::first-line { ... }
15 / 16

Resumen

Las pseudo-clases dan vida y lógica a tu CSS.

1. Interacción: :hover, :focus, :active.
2. Estructura: :nth-child(), :first-child.
3. Formularios: :checked, :disabled, :valid.
4. Lógica: :not(), :has().
5. Usa :focus-visible para mejor accesibilidad.
